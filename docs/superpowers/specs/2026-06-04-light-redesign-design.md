# Светлый редизайн промо-лендинга Laserhouse

Дата: 2026-06-04. Статус: одобрено пользователем (визуальный макет v2 в
`.superpowers/brainstorm/3603-1780586749/content/full-page-b-v2.html`).

## Проблема

Лендинг тёмный (чёрный фон + тёмно-красный), пользователи жалуются «мрачно».
Нужен светлый вариант в фирменном стиле Laserhouse.

## Утверждённое направление

«Beauty: тёплый светлый» (вариант B) + правки v2: больше контраста для
карточек акций, больше нежных градиентов («вау»-эффект).

### Палитра (новые токены)

| Токен | Было (dark) | Стало (light) |
|---|---|---|
| фон | #080506 | перламутр: `linear-gradient(#fdf8f5 → #fbf1ec → #f9ece6)` + персиково-розовые radial-переливы по углам |
| текст ink | #f5efea | #241114 |
| muted | #a89e9b | #8a7a76 |
| faint | #736a68 | #b09a93 |
| красный | #e21d28 (не меняется) | #e21d28 + светлые произв.: коралл #ff7a64, персик #ffb199 |
| карточки/glass | тёмное стекло | белое стекло: `linear-gradient(rgba(255,255,255,.92), rgba(255,244,240,.65))`, рамка rgba(255,255,255,.9), тёплые тени rgba(190,80,80,.25) |

### Ключевые решения по секциям

- **Орбы фона** — вместо тёмно-красных: мягкие розово-персиковые свечения
  (низкая непрозрачность, blur), grain приглушить до ~.03.
- **Nav** — светлое стекло; **логотип возвращается к оригинальному красному**
  (убрать `filter: brightness(0) invert(1)`).
- **Hero** — «ЦІНИ» тёмный градиент (#2a1216→#52242b), «яких більше не
  буде!» — градиент бордо→красный→коралл→персик. Чипы — белое стекло.
- **Таймер** — юниты белое стекло; live-состояние — розоватое стекло с
  красной рамкой; секунды красным.
- **Секция офферов — контрастная полоса**: фон глубже тоном
  (`#f8e6e0 → #f6e0d8` + розовый radial) чтобы белые карточки выстреливали.
- **Карточки акций**: градиентная рамка (перламутр→розовый→коралл, реализация
  через border-box/padding-box на 1.5px), глубокая тёплая тень; красный
  multiply-tint с фото убрать, нижняя кромка фото растворяется в белый;
  бейдж −% со свечением; цена — крупно красным градиентом в розовой
  градиентной плашке.
- **Steps / Conditions / Trust** — белое стекло на перламутровом фоне,
  номера шагов градиент коралл→красный.
- **Карта Украины** — светлая: заливка формы лёгкая красная, обводка красная,
  подписи городов тёмные (убрать белый текст с тёмной обводкой).
- **Final CTA** — переливающаяся градиентная рамка + нежный розовый radial
  изнутри, кнопка-пилюля.
- **Footer** — тёплый градиент `#f7e6e0 → #f3ddd5`, текст тёмный, красные
  заголовки колонок. `fon.png` (mix-blend-mode:screen рассчитан на тёмный
  фон) — переключить на multiply/обычный с низкой непрозрачностью, проверить
  визуально; если плохо — убрать.
- **Мобильный sticky-bar** — светлое стекло.
- **Платёжная модалка** — белая карточка, светлый размытый backdrop
  (rgba тёмного ~.4 + blur), инпуты светлые с красным focus.

### Что НЕ меняется

Весь HTML/контент, тексты, JS-логика (таймер, модалка, scroll-анимации),
шрифты (Cormorant Garamond / Manrope / Oswald), фирменный красный #e21d28,
структура секций. Меняется только CSS (и атрибуты, завязанные на тёмную
тему, если такие есть).

## Реализация

Один файл `index.html` — правится блок `<style>`. Проверка визуальная:
открыть страницу локально, пройти все секции + мобильную ширину (≤560px),
проверить модалку и оба состояния таймера (soon/live — подменой даты в JS
на время теста).
